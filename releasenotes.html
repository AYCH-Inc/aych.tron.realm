TightDB - Release Notes

TightDB build 16

Released on Nov 16th 2012

Improvements:

  • Transaction support (experimental, C++ and Python only).
  • Run-time detection of SSE instruction set.

Bug Fixes:

  • Fixed a memory leak when using Table::create().

C++ API:

  • New method Table::has_shared_spec().
  • Method Table::count() renamed to Table::count_int().
